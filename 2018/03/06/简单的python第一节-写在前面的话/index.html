大家好，今天我想给大家带来的是一个python系列教程。有人会问，现在不论是网络上还是在书店里都能看到很多python教程，为什么还要再写一个呢？的确，随着python应用越来越广泛，各类教程也是多如牛毛，但真正适合完全不懂编程的读者看的却寥寥无几。因此我决定尝试着写一份为从来没有接触过编程语言的人准备的教程。当然，在这过程中很可能会有一些没有解释周全的地方，还请多多包涵，也可以在github上开issue提供意见，感谢各位支持。

为什么选择python

至于这篇教程选择python的原因，主要还是python目前是一门应用广泛而且学习起来比较简单的编程语言。很多人可能接触的第一门语言是C/C或者Java。这些语言可以帮助大家对程序本身运行过程有更多了解，但是它们中存在一些比较复杂的概念（比如C/C中的指针和引用以及Java中的内部类和泛型这类特殊的语法。本篇教程不涉及这些语言因此大家不需要对这些概念有所了解）。这些概念会阻碍我们去实现我们脑中的想法，不能让我们以最快的速度领略到计算机强大的能力，而对于初学者来说这些概念也很难理解，很容易对编程失去信心。因此我选择了近年来在商业和科学上都被广泛运用的python。python入门较为简单，而且在几十年的发展中也吸收了当代编程语言的精华，因此在学会python之后再去学其他语言就会简单很多了。

要看懂这篇教程需要什么呢

扯了这么多，下面就要说说要看懂这篇教程要准备什么了。既然是给完全不懂编程的人读的教程，那么既然我也不会要求有任何对编程语言或者计算机系统的认识，但我还是建议大家能够有以下几个技能。

  • 能够熟练地开关机以及安装软件，不论是在Windows，Mac OS X还是在linux的任何一个发行版上
  • 有基础的英文读写能力
  • 有一颗百折不挠的心，能够自己寻找问题的解决方法

除此之外，如果还能做到以下几点，你的学习之路会更加轻松

  • 能够使用google和stackoverflow(一个大型的程序员交流社区)寻找问题的解决方法，你的问题可能已经被很多人遇到过，也已经被大佬解决了
  • 有初中级别的数学知识，这样理解一些来自数学的概念时会更加轻松

准备好这些，我们就可以来一起认识一下python是什么了。在下一节，我会带领大家了解计算机程序是怎么运行的，而python到底在其中扮演什么角色，请大家多多关注。
