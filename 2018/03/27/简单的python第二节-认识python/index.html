那么，今天我们就开始正式学习python了。大家是不是有些兴奋呢？那么，就让我们一起来看看python到底是什么吧！

python简介

首先，我们先来认识一下这位新朋友。python出生于1989年，是由Guido van Rossum开发的一款编程语言。python的语法十分新颖而又简洁，因此吸引了大量的用户使用。现在，python不仅是作为一款简单方便的计算机语言供一些软件公司使用，同时也在学术和科研领域发挥了很大的用途。可能目前我们还不能完全理解这段话中的一些术语，也不能感受到python的强大，但这都没有关系，我们会在以后慢慢学习到python的方方面面，并亲自写出几个真正的程序。

我们能用python做什么？

由于python能轻松地连接一些用C或者C语言编写的库（使用C和C能够让代码以较快的速度运行），人们也在不断发掘它的用途。下面列举一些python使用的知名案例。

  • google内部的大量测试脚本均使用python编写
  • 豆瓣网和Youtube在最初的时候几乎全部使用python作为支撑其运行的程序
  • 许多神经网络机器学习框架都有python接口供工程师调用，比如mxnet, pytorch和caffe
  • NASA使用python的numpy库和scipy库进行科学计算与建模

虽然目前我们还不知道这些用途到底有什么意义，也不知道什么是测试脚本啊，神经网络啊之类的东西，但我们可以看到，python的使用领域从商业开发到科学计算都有涉及。如果在第一次学习一门语言时可以认识到自己以后可以做这么厉害的东西，甚至能接触到Google的工程师的工具，以后的学习也会多一点兴趣吧。

这篇教程怎么介绍python呢？

python是一门非常复杂的语言，我们这篇教程也只能涉及其中的一小部分。由于使用python开发一个我们正常双击图标就能打开的程序是十分困难的，因此我们将制作一个能在浏览器里运行的程序，也就是我们平时上网浏览的网站了。我将在下一节展示一个小小的网站程序，我们的第一步也就是从这个小网站开始。在最后我们的目标是制作出一个完整的博客程序。那么让我们一起努力吧！
