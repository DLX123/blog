在上一节中，我们认识了python是怎样的一门语言，但是我们仍然没见过一行python代码是长什么样子的。那么今天我们就来看看到底什么样的代码是python。

一些python代码

这一节我想给大家展示一下接下来我们要做的一个小网站，在此之前我认为先把代码贴出来比较能让大家清楚这是一个非常简单的小程序，大家不必过于惊慌。那么我们来看看代码长什么样吧。

main.py

from flask import Flask, render_template, request
app = Flask(__name__)

@app.route('/')
def hello_world():
    return render_template("test.html")

@app.route('/login', methods=["POST", "GET"])
def login():
    if request.method == "POST":
        username = request.form.get("username")
        password = request.form.get('password')
        if username == "admin" and password == "123456":
            return "success"
        else:
            return "failed"
    else:
        return "login"
if __name__ == '__main__':
    app.run()

main.html

<html>
    <head>
        <title>test</title>
        <style>
            #hello {
                color: red;
            }
        </style>

    </head>
    <body>
            <script type="text/javascript">
            function checkform() {
                password = document.getElementById("password").value;
                if (password.length<5){
                    alert("password too short!");
                    return false;
                }
                return true;
            }
        </script>
        <p id="hello">hello world!</p>
        <form method="POST" action="/login" id="loginForm" onsubmit="return checkform()">
            <input name='username' id='username' type='text'>用户名</input>
            <input name='password' id='password' type='password'>密码</input>
            <input name='submit' id='submit' type='submit' value='提交'></input>
        </form>
    </body>
</html>

大家可以看到，我们写的代码包含了两个文件，第一个文件是以py结尾的，这就是python代码，而第二个html是什么好像没有人说过。html实际上就是浏览器使用的语言，我们平时看的网页其实都是html写成的。这篇教程也会在下一节开始开设一个全新的板块教授html，css和javascript(这三门语言是网页编写必须的语言)。放心，虽然看起来多学三门语言十分可怕，但html和css实际上是十分简单易懂的，而javascript和python也比较相似，而且我也会延续简单的讲课风格，相信大家一定能够理解。
